# Hotel Pam Premium - Management System

## Description
This is a console-based hotel management tool built in Java. It allows the user to manage room bookings and guest checkouts efficiently.

## Features
* **Check Availability:** See which rooms are empty.
* **Booking System:** Assign guests to rooms.
* **Checkout:** Calculate bills and release rooms.

## How to Run
1.  Download the `Main.java` and `Room.java` files.
2.  Open them in any Java IDE (IntelliJ, Eclipse) or an Online Compiler.
3.  Run `Main.java`.
